## 🚀 Featured Projects

### 🎙️ [Audio Dataset Manager](https://github.com/maepopi/audio-dataset-manager)
An all-in-one tool to prepare audiobooks and large audio files for TTS and voice cloning pipelines. Built with Python and Gradio.

### 🎨 [Diffusers in ComfyUI](https://github.com/maepopi/Diffusers-in-ComfyUI)
A custom ComfyUI node enabling the use of the Diffusers pipeline directly inside ComfyUI. Available via ComfyUI Manager and Comfy Registry.

### 🤖 [LLM Trainer](https://github.com/maepopi/LLM_Trainer)
A Gradio app to help users quickly and easily fine-tune LLMs. Built as a learning-by-doing project to understand how large language models work under the hood.
